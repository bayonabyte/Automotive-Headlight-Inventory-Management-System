# Automotive Headlight Inventory Management System

Web-based inventory management system developed for an automotive headlight workshop. The system allows product registration, search, update, and deletion.

## Technologies Used
- Python
- Flask
- HTML
- CSS
- SQLite
- Git

## Project Purpose
Small automotive workshops often manage inventory manually or without proper digital tools.  
This project provides a simple, structured, and efficient solution to manage products, improve organization, and reduce operational errors.
